# MyMVC

- 自己动手设计MVC框架（设计模式与PHP高级特性）
- M 数据库
- V 模板引擎
- C 路由

- 基于composer的包管理器
- 路由规则灵活，可以随意自定义路由规则
- 包含默认路由：controller/model/act/...
- 轻量级的ORM,轻松实现CURD
- 轻量而强大的模板引擎，安全过滤等
- 配置文件（数组）
- 数据库迁移采用migration
